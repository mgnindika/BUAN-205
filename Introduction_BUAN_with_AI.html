Predictive Lab — Forecast Next 3 Months

Dataset

Open a blank sheet: sheet.new → Import CSV below.

Download CSV: monthly_sales.csv

  1. File → Import → Upload CSV → Insert new sheet.
  2. Sort by date ascending (A→Z).
  3. Add 3 future months (next rows).
  4. Use =FORECAST.ETS(next_date, sales_range, date_range).
  5. Insert a line chart: Actual vs Forecast.
Expected pattern

Seasonal uptick in Q3; July≈165k, Aug≈172k, Sep≈178k (illustrative).

Reference Visual

Tip: Use data labels on the forecasted points for clarity.

Day 2 · Hands‑on

Prescriptive Lab — Optimize Marketing Spend

Dataset

Open a blank sheet: sheet.new → Import CSV below.

Download CSV: channel_roi.csv

  1. Columns: Channel, ROI. Create Spend cells for each.
  2. Total budget cell (e.g., 100000).
  3. Objective cell: =SUMPRODUCT(spend_range, roi_range)
  4. Constraints: SUM(spend)=Budget; spend ≥0; optional min/max.
  5. Extensions → Add‑ons → Solver → Maximize objective.
Expected result

Approx split: Social 45% · SEM 30% · Email 20% · Print 5% (ROI‑dependent).

Reference Visual

Use data validation to cap spend ranges for realism.

Source: HBR case discussions on surge pricing & trade‑offs.
